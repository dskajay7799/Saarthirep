## 📌 Project Overview
Saarthi Rep is an AI-powered web application designed to provide an interactive and smart user experience.  
It combines voice interaction and modern UI to deliver a simple assistant-style interface that helps users interact naturally with the system.

This project was built using AI tools and frontend technologies to demonstrate a working real-world prototype.

---

## ✨ Key Features
- 🎤 Voice-based interaction (speech input support)
- 🤖 AI-powered responses and processing
- ⚡ Fast and responsive web interface
- 📱 Mobile-friendly design
- 🌐 Fully browser-based (no installation required)
- 🔊 Real-time interaction experience
